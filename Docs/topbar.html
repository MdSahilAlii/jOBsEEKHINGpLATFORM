Topbar Option

To enable the dark topbar, add data-topbar-color="dark" to the <html> tag in your layout. You can apply this dynamically using the useLayoutContext with changeTopBarColor('dark').
To enable the light topbar, add data-topbar-color="light" to the <html> tag in your layout. You can apply this dynamically using the useLayoutContext with changeTopBarColor('light').
