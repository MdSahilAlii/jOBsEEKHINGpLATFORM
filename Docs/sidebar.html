Sidebar Option

To switch to a light sidebar, add data-sidenav-color="light" to the <html> tag in your layout. You can apply this dynamically using the useLayoutContext with changeSideNavColor('light').
To switch to a dark sidebar, add data-sidenav-color="dark" to the <html> tag in your layout. You can apply this dynamically using the useLayoutContext with changeSideNavColor('dark').
To switch to a default sidebar, add data-sidenav-size="default" to the <html> tag in your layout. You can apply this dynamically using the useLayoutContext with changeSideNavSize('default').
To switch to a collapse sidebar, add data-sidenav-size="collapse" to the <html> tag in your layout. You can apply this dynamically using the useLayoutContext with changeSideNavSize('collapse').
To switch to a offcanvas sidebar, add data-sidenav-size="offcanvas" to the <html> tag in your layout. You can apply this dynamically using the useLayoutContext with changeSideNavSize('offcanvas').
